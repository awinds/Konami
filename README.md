## 使用方法

1.下载本插件，放在 `usr/plugins/` 目录中

2.文件夹名改为 `Konami`

3.登录管理后台，激活插件

4.刷新你的页面,来试试敲上密码`↑ ↑ ↓ ↓ ← → ← → B A`
